# 이스터에그 & 숨겨진 인터랙션 목록

## ✅ 구현 완료

### 미션 시스템
- 풋터 "미션" 링크로만 접근 — 일반 방문자는 존재 자체를 모름
- 미션 페이지 7초 카운트다운 (find-mission-page)
- 스크롤 끝까지 내리기 (scroll-to-bottom)
- 언어 토글 버튼 클릭 (toggle-language) — "세계로 열린 문"
- 다크모드 토글 클릭 (toggle-dark-mode) — "밤의 문 / 빛으로 들어가기"
- 프로젝트 메뉴 열기 (open-project-menu) — "기록의 서랍"

### 히든 미션
- **F12** 누르기 → 히든 미션 달성 (데스크탑)
- **패치노트 페이지 방문** → 히든 미션 자동 달성
- 히든 미션 카드 **10초 롱프레스** (모바일)

### 로고 인터랙션 (오늘 추가)
- 설정에서 **강조색 변경** → `93` 무지개 그라데이션 flash
- 설정에서 **폰트 변경** → `93` 슬롯머신 스크램블 (0-9 숫자로)

## 💡 아이디어 (미구현)

### 트리거 기반
- `wansang` 로고 **N번 클릭** 시 효과 (폭죽? 미션 달성?)
- **Konami 코드** (↑↑↓↓←→←→BA) 입력 시
- 검색창에 **특정 단어** 입력 시 반응 (예: "wansang", "easter egg")
- **생일(특정 날짜)** 방문 시 자동 이벤트
- 설정에서 **모든 프리셋을 순서대로** 눌렀을 때

### 시각 반응
- 다크모드 켤 때 별이 떨어지는 효과 (현재는 토스트만)
- 스크롤 속도 빠를 때 헤더 흔들림
- 특정 페이지 첫 방문 시 작은 환영 효과

### 풋터 게임 (장기)
- 딕싯 / 캔트스탑 / 야찌 / 오델로 미니게임
